# 待办事项清单应用 (To-Do List App) 功能说明文档

## 应用概述

基于 Android ListView 的待办事项清单应用，实现了基本的任务管理功能，支持任务查看和删除操作。

---

## 已完成的要求

### 1. 主界面布局 (MainActivity) ✓
- **ListView 作为任务列表容器**
- **列表项显示任务名称和截止日期**
  - 任务名称：黑色字体，18sp，粗体
  - 截止日期：灰色字体，14sp，格式为"截止日期: YYYY-MM-DD"
- **列表项之间有清晰分隔**
  - 使用 1dp 灰色分隔线
  - 每个列表项有内边距，视觉效果清晰

### 2. 数据模型 ✓
- **TaskItem 类包含以下属性**：
  - `taskName`：任务名称 (String)
  - `dueDate`：截止日期 (String)
  - 完整的构造函数、getter 和 setter 方法

### 3. 列表适配器 ✓
- **自定义 TaskAdapter 继承 ArrayAdapter**
- **自定义列表项布局 (list_item_task.xml)**
  - 包含两个 TextView 分别显示任务名称和截止日期
  - 包含分隔线 View
- **正确数据绑定**
  - 在 `getView()` 方法中将数据绑定到列表项视图
  - 实现视图复用机制，提高性能

### 4. 示例数据 ✓
- **使用指定数据初始化列表**：
  1. "Complete Android Homework" - "2025-12-20"
  2. "Buy groceries" - "2024-05-01"
  3. "Walk the dog" - "2024-04-20"
  4. "Call John" - "2024-04-23"

### 5. 基本交互功能 ✓
- **点击任务查看详情**
  - 点击任意任务项弹出对话框
  - 显示任务名称和截止日期完整信息
- **长按任务删除**
  - 长按任意任务项弹出确认对话框
  - 确认后删除任务并刷新列表
  - 提供取消选项防止误操作
- **视觉反馈**
  - 点击列表项有水波纹效果
  - 列表支持平滑滚动

## 功能说明

### 1. 查看任务列表
- 打开应用即可看到所有待办任务
- 任务按添加顺序显示
- 支持上下滑动查看所有任务

### 2. 查看任务详情
- **操作**：轻触（点击）任意任务
- **结果**：弹出对话框显示任务完整信息
- **示例**：点击"Buy groceries"显示：
  ```
  任务详情
  任务名称: Buy groceries
  截止日期: 2024-05-01
  ```

### 3. 删除任务
- **操作**：长按（按住不放）任意任务 2-3 秒
- **结果**：弹出确认对话框
- **选项**：
  - **删除**：确认删除，任务从列表移除
  - **取消**：返回列表，任务保留
- **反馈**：删除成功时有 Toast 提示

### 4. 浏览任务
- **向上滑动**：查看下方任务
- **向下滑动**：查看上方任务
