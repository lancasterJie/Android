# Android 待办事项清单实验报告
## 一、实验目的
- 掌握`ListView`列表展示逻辑
- 学会用Java类封装数据
- 实现自定义`ArrayAdapter`绑定数据与视图

## 二、实验环境
Android Studio + Android模拟器（API26+）

## 三、核心步骤
| 模块          | 功能                          |
|---------------|-------------------------------|
| TaskItem类    | 封装“任务名称+截止日期”属性   |
| 列表项布局    | 用2个TextView显示任务信息     |
| TaskAdapter   | 绑定数据到列表项，复用视图    |
| 主界面        | 初始化示例数据，绑定ListView  |

## 四、实验结果
- 界面：输入框+按钮+任务列表
- 列表展示4条任务，含名称（加粗）和日期（灰色）
- 列表项有分隔线、点击反馈

## 五、总结
完成基础待办列表核心功能，掌握“数据-适配器-视图”分层设计.
