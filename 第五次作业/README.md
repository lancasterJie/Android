# ToDoList 作业项目说明

本项目是一个基于 **Android 原生开发** 的简单待办事项（ToDoList）应用，完成了作业要求的功能，实现了任务的添加、标记、完成和删除。

---

## 功能实现逻辑

1. **任务数据结构**

   * 使用 `TodoItem` 类保存每条任务的信息：

     * `title`：任务标题
     * `time`：任务创建时间
     * `isStar`：是否标记为重要
     * `isDone`：是否完成

2. **界面布局**

   * **输入区域**：EditText + 添加按钮

     * 按钮和输入框之间有间距
   * **任务列表**：RecyclerView 显示任务

     * 每条任务背景为浅灰色，任务之间有间距
     * 任务标题和创建时间纵向排列
     * 星号图标显示重要等级

3. **任务操作**

   * **添加任务**：

     * 点击添加按钮
     * 读取 EditText 内容，如果非空则创建 `TodoItem`
     * 生成当前时间，加入任务列表
     * 刷新 RecyclerView
   * **标星 / 完成 / 删除**：

     * 长按任务弹出菜单：

       1. 标记为重要（切换星号显示）
       2. 标记为完成（任务标题划线）
       3. 删除任务（从列表中移除）
     * 数据修改后调用 `adapter.notifyDataSetChanged()` 刷新界面

4. **UI 状态变化**

   * 标星任务显示星号图标
   * 完成任务标题加横线
   * 删除任务直接从 RecyclerView 移除
   * 每条任务用灰色浅背景区分

---

## 技术实现

* **语言**：Java
* **控件**：

  * RecyclerView + Adapter 实现任务列表
  * LinearLayout 和 TextView 组成每条任务项
* **样式**：

  * 浅灰色背景区分任务项
  * 横线划掉完成任务
* **事件处理**：

  * 点击 EditText 清除 hint
  * 点击添加按钮创建任务
  * 长按任务弹出操作菜单

---

## 总结

本项目通过 RecyclerView + Adapter 完成了基本 ToDoList 功能，能够：

* 添加任务并显示创建时间
* 标记任务重要程度
* 完成任务并划线显示
* 删除任务
* 界面整洁

符合安卓开发作业要求，同时练习了事件监听、Adapter 数据刷新、Drawable 样式和布局优化。
