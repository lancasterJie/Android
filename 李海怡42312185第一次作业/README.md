## 项目概述
本项目是一个实现按钮相关功能的Android应用。

## 实现功能

###  主要功能
- [ ] 功能1：按钮显示跳转
- [ ] 功能2：按钮隐式跳转
- [ ] 功能3：待返回结果的跳转
- [ ] 功能4：带结果跳转实现长按监听器
- [ ] 功能5：带结果返回有取消按钮

## 代码结构说明

```
app/
├── java/com.example.test/
│   ├── MainActivity.java           # 主界面
│   ├── SecondActivity.java         # 跳转的界面
│   └── ThirdAcitivity.java         # 实现带返回结果跳转的界面
├── res/
│   ├── layout/              # 界面布局文件
│   ├── values/              # 资源文件
│   └── drawable/            # 图片资源
└── AndroidManifest.xml      # 应用配置文件
```
